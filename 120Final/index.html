Time Dungeon

This is a game made by Zijing Guo, Curry Xu & Shisong Liu(Skylar Liu) for CMPM120 & ARTG120 during Spring 2019.

This is a quarter long project, a HTML5 game using Phaser CE framework

The game is about helping to free "Mankind" from "the God", you need to guide the warrior, Lazlo, through the dungeon and avoid the traps all over the dungeon. Although the Mankind should have etneral life, but things are operating differently inside the dungeon. Strange thing is awaiting Lazlo at the very bottom of this 5 level dungeon...

Use arrow keys to move, see main.js for more options
